# Flyppy Mosquit
   
## A game about a mosquito in the FLAPPY BIRD format

### Functions:
+ Pop-up windows
+ Time in the game
+ PBest time in the game
+ There is an opportunity to start again
  
  
+ HELP THE MOSQUITO TO SURVIVE
